# 🚀 React CRUD App

A complete CRUD (Create, Read, Update, Delete) application built using React.js and Axios.
This project demonstrates how to perform all CRUD operations with API integration.

## 🔥 Features

* Fetch Posts (GET)
* Add New Post (POST)
* Update Existing Post (PUT)
* Delete Post (DELETE)
* Reusable Form Component
* Dynamic Form (Add / Edit mode)
* Clean UI and State Management

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* Axios
* REST API (JSONPlaceholder)
* CSS
